# FleetGrid Frontend

Modern React dashboard for FleetGrid vehicle optimization system.

## Setup

```bash
npm install
npm run dev
```

## Features

- Dark theme UI
- Interactive Leaflet maps
- Real-time analytics with Recharts
- TypeScript for type safety
- Tailwind CSS for styling

## Environment

Make sure backend is running on `http://localhost:8000`

## Build

```bash
npm run build
npm run preview
```
